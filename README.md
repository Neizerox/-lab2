# Задание 1

## Задача 1

### Текст задачи

На основе списка, содержащего числа от 1 до 7, получить список из соответствующих названий дней недели.

### Алгоритм решения

Рекурсивная функция getlist обеспечивает построчный ввод чисел: она читает строку, проверяет, является ли она пустой (условие завершения ввода), является ли первый символ цифрой, и находится ли число в диапазоне от 1 до 7. При корректном вводе число добавляется в список, при ошибке выводится сообщение и ввод повторяется. Функция toDayName с помощью сопоставления с образцом ставит в соответствие числам от 1 до 7 названия дней недели на русском языке , остальные значения помечаются как "Неизвестно". В функции main происходит ввод чисел через вызов getlist, затем с помощью List.map каждый элемент полученного списка преобразуется в название дня через функцию toDayName. Результат выводится на экран в виде двух списков: исходных чисел и соответствующих им дней недели.

## Задача 2

### Текст задачи

На основе списка, содержащего числа от 1 до 7, получить список из соответствующих названий дней недели.

### Алгоритм решения

Функция countWithKDigits использует List.fold для аккумуляции результата: она проходит по каждому элементу списка, преобразует число в строку, проверяет длину строки, и если она равна k, увеличивает счетчик на единицу. Начальное значение аккумулятора равно 0. В функции main программа запрашивает у пользователя ввод чисел через пробел, преобразует их в список целых чисел, затем запрашивает значение k. После вызова countWithKDigits результат выводится на экран.
